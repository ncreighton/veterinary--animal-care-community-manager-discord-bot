# Veterinary & Animal Care Community Manager Discord Bot

Automated moderation, welcome flows, role management, and engagement analytics for Veterinary & Animal Care communities. Purpose-built for Veterinary & Animal Care community needs.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
